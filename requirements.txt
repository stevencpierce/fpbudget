# infinite retry/recursion loop on every upload. The env-var + Config()

# checksums are off by default. Revisit when R2 supports flexible checksums.

# custom GeventWebSocketNoSSLWorker — the gunicorn worker hangs during
# module import and never binds its port. gevent-websocket (required
# for our socketio transport) has been unmaintained since 2017 and
# doesn't track gevent's API changes past v25. Stay in the 24.x /
